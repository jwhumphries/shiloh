# Shiloh

A custom Hugo theme built with Tailwind CSS v4 and daisyUI.

## Features

- Built with Tailwind CSS v4 (using the new Oxide engine)
- Integrated with daisyUI for component styling
- Responsive design with mobile-first approach
- Dark mode support
- Search functionality
- OKLCH color space for wider color gamut
- Container queries for true responsive components
- Custom utilities and animations

## Development

### Prerequisites

- Docker and go-task
- Or: Hugo (0.151.0+), Bun (latest)

### Local Development

```bash
task dev
```
