Shiloh is a modern Hugo theme designed for blogs, documentation sites, and content-rich websites. It combines cutting-edge CSS technologies with a focus on accessibility, performance, and developer experience.

## Technology Stack

### Core

- [Hugo](https://gohugo.io/) 0.151.0+ - Static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) - Oxide engine with native CSS imports
- [daisyUI v5](https://daisyui.com/) - Component library with custom themes
- [FlyonUI v2](https://flyonui.com/) - Extended UI components

### Client-Side Libraries

- [Fuse.js](https://www.fusejs.io/) - Lightweight fuzzy search
- [theme-change](https://github.com/saadeghi/theme-change) - Theme switching with localStorage
- [Quicklink](https://getquick.link/) - Prefetching for instant navigation
- [Iconify](https://iconify.design/) - Tabler icons via CSS classes

### Build Tools

- [Bun](https://bun.sh/) - Fast JavaScript runtime and package manager
- [Docker](https://www.docker.com/) - Containerized development environment
- [Taskfile](https://taskfile.dev/) - Task runner for build automation

## Design Philosophy

Shiloh prioritizes:

- **Accessibility first** - Semantic HTML, ARIA labels, keyboard navigation, and high contrast colors
- **Performance** - Lazy loading, prefetching, minimal JavaScript, and optimized images
- **Modern CSS** - OKLCH color space for perceptually uniform theming
- **Developer experience** - Docker-based development, hot reloading, and modular configuration

## Inspiration

This theme was heavily inspired by [Congo](https://github.com/jpanther/congo), an excellent Hugo theme by James Panther. Shiloh shares many of Congo's configuration patterns while updating the underlying CSS framework to Tailwind v4, daisyUI v5, FlyonUI, and adding enhanced accessibility features.
