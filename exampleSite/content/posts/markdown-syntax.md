This post demonstrates the various Markdown syntax elements supported by Hugo and the Shiloh theme.

## Headings

Headings from `h2` through `h6` are supported and styled appropriately.

### This is an H3
#### This is an H4
##### This is an H5
###### This is an H6

## Emphasis

You can use **bold text** and *italic text* for emphasis. You can also combine them with ***bold and italic***.

You can use ~~strikethrough~~ text as well.

## Lists

### Unordered Lists

- First item
- Second item
  - Nested item
  - Another nested item
- Third item

### Ordered Lists

1. First item
2. Second item
   1. Nested item
   2. Another nested item
3. Third item

## Links

[This is a link](https://gohugo.io) to the Hugo website.

You can also use [reference-style links][hugo-link].

[hugo-link]: https://gohugo.io

## Blockquotes

> This is a blockquote.
>
> It can span multiple paragraphs and includes beautiful styling in the Shiloh theme.

## Code

Inline `code` has back-ticks around it.

Code blocks use three back-ticks:

```
Plain text code block
No syntax highlighting
```

## Horizontal Rules

Use three or more hyphens, asterisks, or underscores:

---

That's a horizontal rule above this line!

## Conclusion

The Shiloh theme provides beautiful typography for all standard Markdown elements.
