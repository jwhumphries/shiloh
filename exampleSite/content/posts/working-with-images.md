Images are an essential part of any website. This post explains how to work with images in the Shiloh theme.

## Adding Images

You can add images using standard Markdown syntax:

```markdown
![Alt text](/images/photo.jpg)
```

## Image Optimization

The Shiloh theme includes several image optimization features:

- **Lazy loading** - Images load as you scroll
- **WebP support** - Modern image format for better compression
- **Responsive images** - Automatically sized for different screens

## Best Practices

1. **Use descriptive alt text** for accessibility
2. **Optimize file sizes** before uploading
3. **Use appropriate formats**:
   - JPEG for photographs
   - PNG for graphics with transparency
   - WebP for modern browsers
   - SVG for icons and logos

## Figure Captions

Hugo supports figure shortcodes for images with captions.

Remember: Images enhance your content but should be used thoughtfully!
