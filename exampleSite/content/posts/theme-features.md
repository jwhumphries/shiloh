This post showcases all the major features available in the Shiloh Hugo theme.

## Design Features

### Modern UI

Built with Tailwind CSS v4, daisyUI, and FlyonUI for a beautiful, modern interface.

### Dark Mode

Toggle between light and dark themes with smooth transitions.

### Responsive Layout

Perfect display on all devices from mobile phones to large desktops.

## Content Features

### Typography

Beautiful, readable typography with the `article-prose` utility class.

### Code Highlighting

Syntax highlighting for all major programming languages.

### Search

Fast client-side search powered by Lunr.js.

### Table of Contents

Automatic TOC generation for longer posts.

## Navigation

### Main Menu

Easy-to-configure main navigation menu.

### Breadcrumbs

Optional breadcrumb navigation (disabled by default).

### Pagination

Smart pagination for list pages and articles.

## Taxonomies

### Tags

Organize content with tags.

### Categories

Group related content into categories.

## Performance

### Lazy Loading

Images load as you scroll for better initial page load.

### Optimized Assets

Minified CSS and JavaScript for fast loading.

### Prefetching

Quicklink integration for instant navigation.

## Customization

### Color Schemes

OKLCH-based color system for beautiful, perceptually uniform colors.

### Flexible Parameters

Extensive configuration options in `hugo.yaml`.

### Override Partials

Customize any partial template as needed.

## Accessibility

### Semantic HTML

Proper HTML5 semantic elements throughout.

### ARIA Labels

Appropriate ARIA labels for screen readers.

### Keyboard Navigation

Full keyboard navigation support.

## Try It Out!

Explore all these features by browsing through the example posts and testing the various theme parameters.
