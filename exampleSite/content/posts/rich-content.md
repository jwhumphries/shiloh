## Code in Context

When explaining technical concepts, you often need both prose and code:

```bash
# Install Hugo
brew install hugo

# Create a new site
hugo new site my-site

# Add the Shiloh theme
cd my-site
git submodule add https://github.com/username/shiloh themes/shiloh
```

## Lists with Details

Here's a checklist for setting up your site:

1. **Install Hugo** - Download from the official website
2. **Choose a theme** - Shiloh is a great choice!
3. **Configure settings** - Edit your `hugo.yaml`
4. **Create content** - Start writing posts
5. **Deploy** - Use Netlify, Vercel, or GitHub Pages

> **Tip**: Always test your site locally before deploying!
