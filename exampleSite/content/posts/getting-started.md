Welcome to the Shiloh Hugo theme! This post will guide you through the basics of using this beautiful, modern theme built with Tailwind CSS v4, daisyUI, and FlyonUI.

## What is Shiloh?

Shiloh is a lightweight, responsive Hugo theme designed for modern websites. It combines the power of Hugo's static site generation with the flexibility of Tailwind CSS v4's utility-first approach.

### Key Features

- **Tailwind CSS v4** with OKLCH color space for beautiful, perceptually uniform colors
- **daisyUI components** for consistent, accessible UI elements
- **FlyonUI** for enhanced component library
- **Dark mode** support with smooth transitions
- **Responsive design** that works on all devices
- **Fast performance** with optimized assets and lazy loading

## Getting Started

To use this theme, you'll need Hugo installed on your machine. Once you have Hugo ready, you can:

1. Clone the theme repository
2. Configure your site's `hugo.yaml` or `hugo.toml`
3. Start creating content!

### Quick Configuration

Here's a minimal configuration to get you started:

```yaml
baseURL: https://example.org/
title: My Awesome Site
theme: shiloh

params:
  colorScheme: shiloh
  defaultAppearance: light
  enableSearch: true
```
